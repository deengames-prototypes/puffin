Namespace Puffin.Core.Ecs.Components

Classes

AudioComponent

Allows an entity to play an audio file (short or long), optionally at a modified pitch. You should be able to play WAV files and OGG files.

CollisionComponent

Makes an entity collide with solid tiles and other entities that have a CollisionComponent

ColourComponent

Adds a coloured rectangle to an entity.

Component

A base component class. Don't add it to entities directly, it has no functionality To create your own components, create subclasses of Component.

FourWayMovementComponent

Adds a component which makes the entity move in four directions in response to the keyboard. By default, this responds to the WASD and arrow keys; you can change these bindings by changing/adding more bindings in your PuffinGame instance. Note that setting this overrides an entity's velocity.

KeyboardComponent

A keyboard component; provides information about which actions are being pressed. Actions are a simple enum that map to one or more keyboard keys. For examples, see the Puffin documentation or unit tests.

MouseComponent

A component with makes an entity clickable.

OverlapComponent

Makes an entity check for overlap with other entities that have an OverlapComponent.

SpriteComponent

Adds a sprite/image (or spritesheet/tilesheet) to an entity.

TextLabelComponent

Adds a text display to an entity.
