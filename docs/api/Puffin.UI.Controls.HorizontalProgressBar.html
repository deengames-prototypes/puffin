Class HorizontalProgressBar

A horizontal progress bar, such as a health bar; it uses a background sprite, and a colour component for the fill value. Note that this is just a pre-made collection of entities (sprite, colour, etc.). You can still Get them and do things like set the progress-bar colour.

HorizontalProgressBar(Boolean, String, Int32, Int32, Int32, Int32, Int32)

Creates a horizontal progress bar of the specified width (inner width). The progress bar is made up of an image, with the actual progress indicator set by a ColourComponent instance.
